# frequency-inr
Pipeline for Implicit Neural Representations (INRs) on image spectra using frequency domain analysis (FFT).

> ## Credits
> 
> 
> This project is based on the framework provided in the [ICIP 2024 INR Images Tutorial](https://github.com/aegroto/icip-2024-inr-images-tutorial) by **aegroto**.
> The original pipeline for spatial image compression has been extended in this repository to support frequency domain analysis (FFT) and Implicit Representation Learning on image spectra.
